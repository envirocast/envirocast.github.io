# EnviroCast — Quantum-Enhanced Environmental Insights
A statically exported Next.js site (App Router) deployed on GitHub Pages at **https://envirocast.github.io**.

## Tech
- Next.js 14 (App Router) — static export (`next export`)
- Tailwind CSS
- Framer Motion (animations)
- Recharts (graphs)
- React Three Fiber + drei (light 3D quantum viz)
- Minimal shadcn-style UI primitives
